Описание проекта

Полнофункциональный
VPN-клиент без App Store

📱

PWA — установка без магазинов

Progressive Web App устанавливается в один клик на iOS и Android, минуя App Store и Google Play. Service Worker обеспечивает оффлайн-работу и мгновенную загрузку интерфейса.

🔐

WireGuard · OpenVPN · VLESS

Готовность к интеграции с любым протоколом: WireGuard, OpenVPN TCP/UDP, VLESS, Shadowsocks. Архитектура позволяет подключить любой backend в течение дня.

⚡

60 FPS на бюджетных устройствах

Чистый HTML5/CSS3/JS без тяжёлых фреймворков. Core Web Vitals: загрузка < 1.5 сек. Chrome, Safari, Firefox, Samsung Internet. От 320px до 4K.

🎨

UI/UX Cyber-Premium 2026

Glassmorphism, анимированная кнопка-орбита, реал-тайм дашборд скорости, drawer-меню выбора серверов (Германия, Нидерланды, США, Сингапур). Dark/Light режим.

🛡️

Kill Switch · DNS-over-HTTPS

WebRTC leak protection, Split tunneling, Auto-connect при старте, DNS-over-HTTPS против утечек. Настройки протоколов на лету.

🌐

White Label · Мультиязычность

Готовность i18n: RU/EN/DE/ES. Нет зависимостей от внешних CDN — 100% автономность. Готов к деплою на Netlify, Vercel, GitHub Pages, VPS.

Специализированное решение

L2 Bridge для диагностики авто

Remote Diagnostic VPN (L2 Bridge)

Готовое решение для удалённой диагностики автомобилей и промышленного оборудования. Современный UX/UI в стиле BRAND TUN взамен устаревших окон. Нативный L2-туннель через UDP, полная поддержка драйвера Wintun. Решена проблема поиска интерфейса 172.31.20.2.

  • ✓ ODIS, Xentry и любые USB-сканеры через проброс
  • ✓ Один .exe файл без лишних зависимостей
  • ✓ Стабильная работа, решена проблема потери соединения
  • ✓ Идеально для сервисов Volkswagen, Mercedes, BMW

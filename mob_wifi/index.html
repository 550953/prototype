01 Постановка задачи

Почему «бытовуха» не работает

Стандартные потребительские решения имеют три системных недостатка, делающих их непригодными для профессиональной эксплуатации.

01
Температурный шок

Li-Ion вздувается на солнце (+70°C), теряет до 40% ёмкости при морозе. Обычный диапазон: 0…+45°C.

02
Конфликты питания

Переключение между зарядкой и отдачей даёт просадки напряжения, приводящие к перезагрузке роутера.

03
Нет мониторинга

Система «умирает» без предупреждения. Нет данных по заряду, температуре, потреблению в реальном времени.

02 Системная архитектура

Система строится по топологии «Общая шина питания» с интеллектуальным управлением нагрузкой через ESP32.

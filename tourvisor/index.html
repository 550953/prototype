Tourvisor App

Структура интерфейса и навигации мобильного приложения

1. Главный экран (Home / Поиск) — точка входа

Верхняя панель
Логотип + поисковая строка (быстрый переход)
Кнопка профиля / авторизации
Переключатель режимов: Турист / Турагент (после логина)
Центральная поисковая форма (основной элемент)
Город вылета (список + аэропорты)
Направление (страна → курорт → город + поиск по карте)
Даты вылета (интерактивный календарь с мин. ценами и расписанием)
Длительность (2–28 ночей)
Состав туристов (взрослые + дети с возрастами)
Кнопка «Найти» (большая, яркая)
Быстрые разделы / табы
Туры (с перелётом)
Отели (только размещение)
Горящие туры
Минимальные цены

2. Экран результатов поиска (Results)

Верх: количество результатов + сортировка (цена, рейтинг, популярность)
Фильтры (выезжающая панель или чипсы)
Базовые: цена, звёзды, питание, пляж, расстояние до моря
Расширенные (только для агентов): кол-во комнат/спален, площадь номера, ключевые слова (swim-up, джакузи…), авиакомпании
Список карточек предложений
Фото отеля + галерея
Название отеля + рейтинг + питание
Цена за человека / за номер + % скидки (last-minute)
Оператор / поставщик
Кнопки: «Подробнее», «В подборку», «Сравнить»
Возможность сравнения нескольких туров

3. Экран детальной карточки предложения (Detail)

Галерея фото отеля и номеров
Полное описание + встроенные характеристики номера
Блок сравнения цен (таблица от разных операторов и сервисов)
Дополнительная информация (рейтинги, отзывы, погода, карта)
Блок действий:
«Забронировать онлайн» (переход к партнёру)
«Отправить заявку в агентство»
«Где купить» → Карта партнёров (2157+ офисов)
«Добавить в коллекцию»
«Поделиться» (мессенджеры / соцсети)

4. Личный кабинет / Профиль (Profile)

Общие функции (для всех)
Мои коллекции / подборки (создание, редактирование, шаринг)
Избранное
История поисков
Настройки (уведомления, валюта, язык)
Профессиональные функции (только турагенты)
Создание брендированных подборок (логотип, цвета)
Фид горящих туров (реал-тайм last-minute)
Подписки на алерты (Telegram-бот)
Генератор репостов (картинка + текст за 2 клика)
Lead-менеджер (работа с заявками)

5. Дополнительные экраны / разделы

Календарь туров (цены по датам из любого города)
Карта «Где купить» (все партнёры с адресами)
Поддержка (чат / телефон / email)
О приложении / Политика конфиденциальности

Нижняя навигационная панель (Bottom Bar)

Поиск (домой)
Результаты / Горящие
Коллекции
Карта / Календарь
Профиль
